
# x) Reading Summaries

## Schneier – Applied Cryptography, Chapter 1 (Foundations)

Key points:

- Cryptography protects:
  - Confidentiality
  - Integrity
  - Authentication
  - Non-repudiation
- Encryption converts plaintext into ciphertext using keys.
- Security must rely on secret keys, not secret algorithms.
- Two main cryptography models:
  - Symmetric key cryptography
  - Public key cryptography
- Systems fail more often because of bad implementation than weak algorithms.
- Common attack models:
  - Ciphertext-only
  - Known-plaintext
  - Chosen-plaintext

## Karvinen 2023 – PGP with GPG

Key points:

- GnuPG implements PGP public-key encryption.
- Users generate:
  - Public key (shared)
  - Private key (secret)
- Public key encrypts messages.
- Private key decrypts messages.
- Digital signatures verify sender identity.
- Encryption + signing ensures confidentiality and authenticity.
